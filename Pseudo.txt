PseudoCode for Gallery on Portfolio

When Next is clicked:
Current container height:0
next container height: 100

When Back is clicked:
Current container height:0
previous container height: 100

How to complete:
Place each container div in array

Function next:
Set current array elements height to 0 and next in array to 100
Function back:
Set current array element height to 0 and previous in array to 100
